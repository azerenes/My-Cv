## Hakkımda

Kırıkkale Üniversitesi **Kontrol ve Otomasyon Teknolojisi** programında eğitim görmekteyim. Mesleki eğitim hayatıma Abidinpaşa Mesleki ve Teknik Anadolu Lisesi Elektrik Pano Montörlüğü dalında başladım. Pano kurulumu, zayıf akım sistemleri ve teknik arıza çözümleri başta olmak üzere saha deneyimine sahibim.

Yazılım geliştirme alanına son dönemde adım atmış olmakla birlikte, kendimi sürekli geliştirmeye açığım. C, Python ve modern mobil/web teknolojileriyle gömülü sistemler, IoT ve yapay zeka odaklı projeler geliştiriyorum.
