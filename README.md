## 🧑‍💻 Profesyonel Özet

Kırıkkale Üniversitesi **Kontrol ve Otomasyon Teknolojisi** programında eğitim görmekteyim. Abidinpaşa MTAL Elektrik Pano Montörlüğü dalından mezun oldum. Pano kurulumu, zayıf akım sistemleri ve teknik arıza çözümleri başta olmak üzere saha deneyimine sahibim.
